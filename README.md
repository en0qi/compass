# compass
天体望遠鏡をセッティングする際、真北（磁北ではなく）と水平がサクッと分かるといいなと思って作った。

## 説明
天体観測時の真北・水平検出アプリを作成しました。天体望遠鏡を設置する際は、磁北ではなく真北を知る必要があり、同時に水平を取る必要があります。暗所で明るい画面を見ると瞳孔が閉じて星が見えなくなってしまうので、黒背景に赤い文字の天体観測時に最も適した画面を作りました。
